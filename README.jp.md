インストール
---

### Npm

	npm install jquery.plugin.starter-kit


使い方
------
	<script src="js/jquery.js"></script>
	<script src="js/jquery.plugin.starter-kit"></script>
	<script>
		$(function(){
			$('body').test({});
		});
	</script>


Options
------

| option name| type | Descriptions |default
|:-----------|:------------|:------------|:------------|
| className | `String` | クラス名 |`'test'`
| num | `Number` | 個数 | `10`
